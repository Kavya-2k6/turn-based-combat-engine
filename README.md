# Turn-Based Combat Engine

A simple console-based turn-based combat game written in Java.

## Features
- Player vs enemy turn-based combat
- Normal attack and special attack
- Mana system for special attacks
- Healing with limited uses and cooldown
- Basic enemy AI logic

## Tech Stack
- Java
- Object-Oriented Programming (OOP)
- Console input/output

## Purpose
This project is built for learning Java, OOP concepts, and Git/GitHub through incremental development.
